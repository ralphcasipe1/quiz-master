# Quiz Master

## Technologies
- Server
  - Ruby on Rails
- Client
  - ReactJS
  - Redux
  - Jest

## Server

```
cd server/
```

```
bundle
```

Create the databse first
```
bin/rails db:create
```

Migrate the tables
```
bin/rails db:migrate
```

Start the Server
```
bin/rails server
```

## Client

```
cd client/
```

For NPM:
```
npm i
```

For Yarn:
```
yarn
```

Run the development
```
npm start
```
