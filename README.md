# Quiz Master

## Technologies
- `server/`
  - Ruby on Rails
- `client/`
  - ReactJS
  - Redux
  - Jest

## Server

Go to the `server/` directory
```
cd server/
```

Install the gems
```
bundle
```

Create the database first
```
bin/rails db:create
```

Migrate the tables
```
bin/rails db:migrate
```

Start the Server
```
bin/rails server
```

## Client

Go the `client/` directory
```
cd client/
```

For NPM:
```
npm i
```

For Yarn:
```
yarn
```

Run the development
```
npm start
```
